# RF 07

## Nome 
 O usuário escolherá a sua próxima ação após finalizar cada fase intermediária do jogo

## Descrição 
  Com a finalização das fases 1,2 e 3 será mostrado para o usuário as suas opções de ações a serem realizadas.

## Atores
- Usuário autenticado

## Prioridade
Essencial

## Entradas e pré-condições
- O usuário selecionará qual opção ele almeja realizar: continuar(próxima fase) ou sair para o menu principal.

## Saídas e pós-condições
- O usuário iniciará a próxima fase ou sairá para o menu principal.

## Fluxos de eventos 

## Fluxo principal
Será mostrado as opções para serem selecionadas:
1. Continuar/próxima fase
1.2 sair para o menu principal

## Fluxo secundário 
1. Caso o usuário não selecione nenhuma das opções, ele irá ficar nessa tela de menu pós-fases.
