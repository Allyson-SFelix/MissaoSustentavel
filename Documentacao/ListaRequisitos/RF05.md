# RF 05

## Nome
Usuário conseguiu encher seu depósito móvel de lixo e buscou o próximo objetivo, esvaziá-lo no centro de coleta.

## Descrição
O usuário atingiu o objetivo primário de cada fase e agora busca avançar a fase, que será no centro de coleta, para realizar a reciclagem nos espaços devidos.

## Atores
- Usuário autenticado

## Prioridade
Essencial

## Entradas e pré-condições
- Movimentação do personagem pelo usuário via teclado para chegar no ícone do centro de coleta.

## Saídas e pós-condições
- Após ir para o centro de coleta, mudará para a interface do centro de reciclagem.

## Fluxos de eventos

### Fluxo principal
1. Após encher a sacola de lixo, aparecerá uma mensagem informando para o usuário deslocar o personagem para o centro de coleta para realizar o descarte adequado para cada item coletado.
2. Um ícone do centro de coleta aparecerá no mapa para o usuário se guiar.
3. Quando o personagem for deslocado para o ícone do centro de coleta, será carregada uma nova interface na tela, simbolizando o centro de reciclagem.

### Fluxo secundário
1. Caso o usuário não se desloque para o ícone da coleta de reciclagem, o jogo não será penalizado, mas também não avançará na fase.
