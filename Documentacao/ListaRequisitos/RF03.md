# RF 03

## Nome  
Usuário clicou para inicializar o jogo

## Descrição  
O sistema receberá a solicitação do botão iniciar o jogo clicado pelo usuário e aparecerá a interface inicial do jogo

## Atores  
- Usuário autenticado  

## Prioridade  
Essencial  

## Entradas e pré-condições  
- Usuário clicou no botão do menu de iniciar o jogo 

## Saídas e pós-condições  
- Será mostrado as instruções do usuário para jogar, contexto e título da fase por 5 segundos. 

## Fluxos de eventos  

### Fluxo principal  
1. Aparecerá texto :
1.1 A fase do jogo explicando sobre a numeração da fase atual (fase 01) 
1.2 A instrução de jogabilidade relacionado com a movimentação do personagem do jogo (todas as fases), sendo:
    1.2.1 “w” para subir
    1.2.2 “s” para descer
    1.2.3 “d” para direita
    1.2.4 “a” para esquerda
1.3 O contexto explicando que resolvendo esse objetivo está ajudando a aplicar determinada ODS
