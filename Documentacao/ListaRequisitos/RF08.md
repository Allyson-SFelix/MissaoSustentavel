# RF 08

## Nome
Usuário reiniciará a fase ao encostar no boss

## Descrição

Quando o personagem do usuário for atingido pelo boss, o jogo exibirá opções para o jogador escolher entre reiniciar a fase do início ou voltar ao menu principal.

## Atores

* Usuário

## Prioridade

Essencial

## Entradas e pré-condições

* Usuário autenticado e em gameplay.  
* O boss encosta no personagem do usuário.  
* Usuário seleciona uma das opções apresentadas (reiniciar ou menu).

## Saídas e pós-condições

* Se o usuário escolher **Reiniciar**, a fase recomeça do início.  
* Se o usuário escolher **Menu**, o sistema retorna ao Menu Principal.

## Fluxos de eventos

### Fluxo principal

1. O boss encosta no personagem do usuário.  
2. O sistema exibe a tela de game over com as opções:  
   * **Reiniciar a fase**  
   * **Voltar ao Menu Principal**  
3. O usuário seleciona uma das opções.  
4. O sistema executa a ação correspondente:  
   * Reinicia a fase do início **ou**  
   * Retorna ao Menu Principal.

### Fluxo secundário

1. Se o usuário não selecionar nenhuma opção, o jogo permanecerá na tela de game over até que uma escolha seja feita.
