# Missão Sustentável (Protótipo Pygame)

## Requisitos
- Python 3.10+
- `pip install pygame`

## Como executar
```bash
cd missao_sustentavel
python -m pip install pygame
python main.py
```

## Controles
- **WASD/Setas**: mover
- **E**: pegar item (quando sobre o item)
- **Q**: descartar na lixeira (quando encostando na lixeira)
- **R**: reiniciar fase
- **ESC**: sair

## Fases 
- Fase 1: 1 lixeira geral (meta 5)
- Fase 2: orgânico + plástico (meta 8)
- Fase 3: papel + vidro + metal (meta 10)
- Fase 4: mix e inimigo que persegue (meta 12)
